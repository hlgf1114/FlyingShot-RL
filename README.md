# FlyingShot-RL

Abstract
-------------
This is a project to make and evaluate a game named FlyingShot.

Make DQN with 3 conv layer and 2 fully connected layer.

### Applied technique
> * Dueling Network
> * Experience Replay
> * Frame Skipping & Stacking
> * Target Network




Requirements
-------------
* tensorflow-gpu=1.15.0
* pygame=1.9.6
* numpy
* matplotlib
* openCV
